# 🗳️ BD 2026 Election Data Collection & Analysis

## 📌 Project Objective

This project focuses on **collecting and analyzing Bangladesh Election 2026 data**.  

- Data collection is performed using **web scraping with Playwright**
- Includes **button click automation and pagination handling**
- Data processing and analysis are performed using **Jupyter Notebook**

The goal is to build a structured dataset and perform meaningful election analysis.
